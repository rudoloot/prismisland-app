# 기지 건설 (Base & Building)

## 1. 왜 포탑은 '직접 탑승형'인가? (기술적 한계 설정)

* 플레이어의 기지는 발전을 통해 기본적인 전기를 생산하고 기초적인 전선과 회로도 수준까지는 구현할 수 있음 (전등을 켜거나 기초적인 가공 기계를 돌리는 정도).
* 그러나 **움직이는 적을 자동으로 탐지, 조준, 격발하는 고성능 마이크로칩(반도체)이나 자동 제어 소프트웨어를 연산할 기술력은 전무함.** 
* 따라서 전기를 동력원으로 사용하는 강력한 포탑을 제작하더라도, **조준과 사격 자체는 인간이나 수인 동료가 직접 탑승하여 레버를 조작하고 트리거를 당겨야만 함.**
